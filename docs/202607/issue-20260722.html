Go技术生态

Go 密码学维护者放大招：把 Passkey 存成一行字符串，还顺手为 Go 1.28 写好了 API
Passkey（无密码登录）是应对网络钓鱼的终极方案，但其服务端实现一直因数据结构复杂、各家存储方案互不兼容而令开发者头疼。近日，Go 官方密码学库维护者 Filippo Valsorda 抛出了一项重磅提案：借鉴密码哈希（如 bcrypt）的成熟经验，将繁杂的 WebAuthn 凭证抽象 并压缩为一行标准化的不透明字符串（passkey record），实现数据库无缝迁移与互操作。基于此格式，他还起草了一份有望并入 Go 1.28 标准库的 crypto/passkey 无状态 API，极大地简化了开发者的接入成本。此外，他还提出了一个反直觉的安全洞见：只要不将 Credential ID 作为数据库主键建索引，就能从根本上免疫 ID 碰撞攻击。该提案为大厂与个人开发者提供了一条兼顾极简与安全的 Passkey 落地新路径。

Twitter之父再出手：Block开源Buzz，要让人类和AI Agent「同工同权」
Block（原Square）7月22日开源发布协作平台Buzz——一个基于Nostr协议、让人类员工与AI Agent在同一工作区内以「同等身份」协>同工作的系统。Buzz为每个Agent分配独立的密码学身份，使其能够像正式团队成员一样发消息、评审代码、跑自动化流程、参与语音会议，所有动>作均被签名并可追 溯，从而解决了「多个Agent共用一个账号、责任无法追溯」的老问题。项目采用Apache-2.0协议开源，支持自托管，也提供Block官方托管服务，同时兼容Claude Code、Codex、goose等主流Agent框架。分析认为，这既是应对「Agent数量爆炸后协调成本失控」的工程解法，也 是Dorsey与Sequoia的Roelof Botha联合提出的「用AI替代科层制」组织理论的具体实践。目前Buzz仍处于早期阶段，relay架构尚未真正去中心化， 移动端和部分工作流审批机制仍在建设中。
